Thoth lab¶

A laboratory providing helper functions for experiments on software stack quality in Jupyter notebooks. This library is used in project Thoth.

See thoth-station/jupyter-notebook and thoth-station/notebooks repos for more information about usage of this library in project Thoth.

Installation¶

This project is released on PyPI, so the latest release can be installed via pip or Pipenv as shown below:

pipenv install thoth-lab
